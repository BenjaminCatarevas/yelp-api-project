Yelp Restaurant Selector

This is a program that gives users a chance to choose restaurants based on certain preferences.

The following choices available:

  • Food type (Mexican, Chinese, etc.)
  • Number of results (max 50, default 10)
  • Location (town, state) - Required
  • Radius to search (max 25 miles, default 10 miles)
  • Restaurants that are currently open (unchecked means all restaurants)
